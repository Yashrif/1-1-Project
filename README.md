# Diary &amp; To-Do App

>**A terminal based diary & to-do app created only using c++. Easy to use. User can add tasks to the to-do list, reminders & daily events to the diary.**

## Keybindings

- Up/down arrow key to scroll
- Enter key to select
- Esc key to back/exit
- D key to delete
